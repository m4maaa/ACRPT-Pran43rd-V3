# ACRPT-Pran43rd V3

เว็บแสดงเกณฑ์ทดสอบสมรรถภาพร่างกายสำหรับเจ้าหน้าที่โดยเฉพาะ แสดงเฉพาะเกณฑ์ผ่านขั้นต่ำ 70% และคะแนนเต็ม 100% ออกแบบแบบ mobile-first และไม่บันทึกข้อมูลผู้ใช้งาน

## การใช้งาน

เปิด `index.html` ด้วยเว็บเบราว์เซอร์ จากนั้นเลือกสถานีและอายุผู้เข้ารับการทดสอบเพื่อดูเกณฑ์ 70% และ 100%

## หมายเหตุ

ควรตรวจสอบข้อมูลเกณฑ์ทั้งหมดกับเอกสารทางการก่อนนำไปใช้งานจริง
